# 前端逻辑流工程

## 启动
```bash
pnpm install
npm run dev
```

## 项目结构
```
.
├── packages
│   ├── compiler        # 编译器
│   ├── designer        # 设计器
│   ├── excuter         # 执行器
│   ├── node-contexts   # 节点上下文
│   ├── node-metas      # 节点定义
│   └── node-types      # 类型定义
├── src
│   ├── App.tsx         # demo
│   ├── main.tsx        # demo 入口

```
